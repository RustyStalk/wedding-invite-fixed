- `script.js` — логика формы.
- `api/send-rsvp.js` — серверный обработчик для отправки ответов в Telegram.

## Как запустить сайт локально
Это просто для просмотра дизайна у себя на компьютере.

### Вариант 1 — очень просто
1. Распакуйте архив.
2. Откройте файл `index.html` двойным кликом.
3. Сайт откроется в браузере.

> Важно: форма отправки в Telegram в таком режиме работать не будет, потому что для неё нужен серверный обработчик.

## Как сделать, чтобы ответы приходили в Telegram
Лучший вариант — разместить сайт на **Vercel**.

### Шаг 1. Создайте Telegram-бота
1. Откройте Telegram.
2. Найдите `@BotFather`.
3. Отправьте команду `/newbot`.
4. Задайте имя и username бота.
5. Сохраните токен, который пришлёт BotFather.

Пример токена:
`1234567890:AA...`

### Шаг 2. Подготовьте GitHub
1. Создайте аккаунт на GitHub, если его нет.
2. Создайте новый репозиторий, например `wedding-invite`.
3. Загрузите в него все файлы из этой папки.

### Шаг 3. Загрузите сайт на Vercel
1. Зайдите на [https://vercel.com](https://vercel.com)
2. Войдите через GitHub.
3. Нажмите **Add New Project**.
4. Выберите ваш репозиторий `wedding-invite`.
5. Нажмите **Deploy**.

### Шаг 4. Добавьте переменные окружения
После создания проекта откройте:
`Project → Settings → Environment Variables`

Добавьте две переменные:

#### 1) TELEGRAM_BOT_TOKEN
Значение: токен вашего бота от `@BotFather`

#### 2) TELEGRAM_CHAT_ID
Значение:
`826467625`

### Шаг 5. Сделайте Redeploy
После добавления переменных:
1. Откройте вкладку **Deployments**
2. Нажмите **Redeploy**

После этого форма начнёт отправлять ответы в Telegram.

## Как проверить
1. Откройте сайт по ссылке Vercel.
2. Заполните форму тестовыми данными.
3. Нажмите отправить.
4. В Telegram должно прийти сообщение с данными гостя.

## Важно
**Не вставляйте токен Telegram-бота прямо в HTML или `script.js`.**
Его смогут увидеть все гости через код страницы.
Токен должен храниться только в переменных окружения Vercel.
